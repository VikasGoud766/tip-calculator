# Tip Calculator 💰

This is a simple web-based tip calculator built using Python (Flask), HTML, and CSS.

## Features
- Calculate tip easily
- Split bill among people
- Simple UI

## Tech Stack
- Python (Flask)
- HTML
- CSS

## How to Run
1. Install Flask:
   pip install flask
2. Run:
   python app.py
3. Open browser:
   http://127.0.0.1:5000/
